# nwputhesis-fonts

Windows 字体文件，用于 nwputhesis LaTeX 模板。

## 字体列表

| 字体 | 文件 | 用途 |
|-----|------|-----|
| 宋体 | simsun.ttc | 中文正文 |
| 黑体 | simhei.ttf | 中文标题 |
| 楷体 | simkai.ttf | 中文斜体 |
| 仿宋 | simfang.ttf | 中文特殊格式 |
| Times New Roman | times*.ttf | 英文正文 |
| Consolas | consola*.ttf | 代码等宽字体 |

## 使用方法

作为 nwputhesis 的子模块使用：

```bash
git clone --recursive https://github.com/1195343015/nwputhesis
```

或

```bash
git clone https://github.com/1195343015/nwputhesis
cd nwputhesis
git submodule update --init
```
